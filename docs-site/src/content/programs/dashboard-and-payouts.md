---
title: Dashboard & Payouts
description: What the recruiter dashboard should show and how recruiter earnings should be presented.
---

The recruiter dashboard is where approved recruiters track squad growth and earnings.

## Dashboard metrics

The dashboard should show:

- recruiter code
- share link
- linked creators
- linked traders
- pending earnings
- lifetime earnings
- recent squad activity
- leaderboard position when available

## Squad view

Recruiters should be able to understand who is in their squad without exposing unnecessary private information.

Useful views include:

- creator count
- trader count
- recently joined users
- active creators
- active traders
- graduated linked creators

## Share tools

Recruiters should have easy share actions for:

- X
- Telegram
- Discord
- direct copy link

## Payout direction

Recruiter earnings should be transparent and easy to audit.

The UI should distinguish:

- estimated earnings
- pending earnings
- claimable or payable earnings
- lifetime earnings

## Leaderboards

Recruiter leaderboards can create social competition.

Useful leaderboard formats:

- weekly top recruiters
- top squad builders
- top creator recruiters
- top trader recruiters
- highest squad activity

Leaderboard moments are good content for X, Telegram, and Discord.
