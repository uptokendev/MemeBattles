Prepare Mode opens on **May 12**.

Prepare Mode is the pre-live phase where the battlefield gets loaded before the switch is flipped.

Creators can prepare campaigns. Recruiters can build squads. Traders can position themselves early. The platform can onboard users before live launches begin.

## Why Prepare Mode exists

A cold launch creates chaos.

Prepare Mode lets MemeWarzone start with:

- creators already queued
- recruiter squads already forming
- campaign pages already prepared
- communities already briefed
- less confusion on launch day

It also gives users time to understand the rules before live trading starts.

## What creators can do

During Prepare Mode, creators can prepare campaign assets and launch plans before live deployment.

Creators should prepare:

- token name and ticker
- logo and banner
- description and narrative
- official links
- launch plan
- community channels
- recruiter relationship if applicable

When MemeWarzone flips live, prepared creators can deploy their campaigns into the active warzone.

## What traders can do

Traders can use Prepare Mode to understand the platform before capital is live.

Traders should learn:

- how the bonding curve works
- how graduation works
- how fees are routed
- what League categories exist
- how reward drops work
- how to avoid scam links

## What recruiters can do

Recruiters can use Prepare Mode to build their squad before trading opens.

Recruiters should:

- apply for the Recruiter Program
- get approved
- share their recruiter link
- onboard creators and traders
- prepare their squad before live deployment

Read: **[Recruiter Program](/programs/recruiter-program)**.

## Prepare Mode vs Live Mode

| Mode | What happens |
| --- | --- |
| Prepare Mode | Campaigns and squads are prepared, but the full battle is not live yet. |
| Live Mode | Creators deploy campaigns, trading opens, Leagues and reward systems become active. |

## Important rule

The live switch should be treated as a serious transition.

Once a campaign is deployed live, it should not casually revert back to preparation status. This protects traders, creators, and platform trust.
