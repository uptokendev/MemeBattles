---
title: League Categories
description: The five main MemeWarzone League categories and what each one rewards.
---

MemeWarzone has five core League categories.

Each category rewards a different type of battlefield performance.

## Perfect Run

Perfect Run rewards campaigns that achieve a clean run based on strict campaign behavior rules.

Direction:

- monthly-only category
- jackpot-style potential
- rollover if no campaign qualifies
- designed to reward rare, clean launch execution

## Fastest Finish

Fastest Finish rewards campaigns that graduate the fastest.

Important rule direction:

- creator buys should be excluded from the measurement
- the category should focus on real external campaign demand

## Biggest Hit

Biggest Hit rewards the biggest single buy during the bonding-curve phase.

This category creates dramatic moments around major entries.

## Top Earner

Top Earner rewards the strongest trader performance.

The intent is to track meaningful trader earnings within the defined League rules and epoch.

## Crowd Favorite

Crowd Favorite rewards campaigns that win the attention battle through UpVotes.

This is the most community-driven category.

## Why multiple categories matter

A single leaderboard usually favors whales.

Multiple categories create more ways to win:

- clean launch execution
- speed
- conviction
- trading skill
- community support

This makes the warzone more dynamic than a simple volume leaderboard.
