## Quick summary

Warzone BNB Airdrops are funded by unlinked recruiter and squad slices.

The weekly airdrop pool is split:

| Program | Share |
| --- | ---: |
| Trader airdrop | 50% |
| Creator airdrop | 50% |

Eligibility uses capped activity, weighted random selection, cooldowns, and anti-abuse checks.
